Ti si senior frontend developer. Izgradi produkcijski spremnu, ali jednostavnu web aplikaciju.

🎯 CILJ PROJEKTA

Izraditi web aplikaciju za studentski festival (UNIFEST) koja generira “sidequestove” (gamified challenges) za korisnike tijekom festivala.

Aplikacija mora biti:

mobile-first (primarno korištenje na mobitelu)
ultra brza (korisnici su na festivalu, često pijani / u pokretu)
bez backenda
bez login sustava
100% frontend (localStorage dozvoljen)
optimizirana za shareanje sadržaja na društvene mreže
🧠 CORE CONCEPT

User dobiva random “quest card” koji sadrži:

title
description
rarity (common / rare / epic / legendary)
XP reward
mode (solo / duo / squad)

User može:

generirati novi quest
označiti quest kao completed
dobiti “quest completion screen”
snimiti sliku (kamera)
shareati rezultat
📦 TECH REQUIREMENTS

Koristi:

React + Vite
TailwindCSS za styling
Web APIs:
navigator.mediaDevices.getUserMedia (kamera)
Canvas API (image generation + watermark)
navigator.share (Web Share API)
localStorage (XP + completed quests)

Bez:

backend-a
database-a
auth sustava
